# vi


## move line
|Command|Description|
|:---|:---|
|0|行頭|
|＾|行頭|
|$|行末|
|%|対応する括弧に移動|
|w|次の単語 (Word)|
|b|前の単語|

## move file
|Command|Description|
|:---|:---|
|gg|最初の行|
|88G|88行目|
|G|最終行|
|H|画面上の最初の行 (Home)|
|M|画面上の中央の行 (Middle)|
|L|画面上の最後の行 (Last)|
|Ctrl + u|半画面上 (Up)|
|Ctrl + d|半画面下 (Down)|
|zz|カーソルが画面中央になるようにスクロール|

## others
|Command|Description|
|:---|:---|
|f(文字)|カーソルがある行の(文字)に移動 (Find)|
|F(文字)|カーソルがある行の(文字)に移動(逆向き)|
|Ctrl + o|古いカーソル位置に戻る。 (Old)|
|Ctrl + i|新しいカーソル位置に進む。|
